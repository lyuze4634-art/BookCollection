这玩意是个简单的PHP图书管理系统


这是一个私人图书管理系统项目，核心目标是给家里的书拍照并上传到网站，保存书名、作者、封面和简介，方便自己和朋友查看。
目前项目的主要功能包括：
密码进入网站：只有输入固定密码才能进入
图书上传：上传封面图，填写书名、作者、简介
图书列表：显示所有书的封面、书名、作者
图书详情页：查看完整信息、大图，支持修改和删除
搜索功能：按书名或作者搜索图书
基于PHP+MySQL+HTML/CSS的小型个人图书整理网站。
接下来有时间的话可能会增加一个收藏系统，比如建一个新表，记录点了收藏的书，然后在另一个界面显示出来。


これはシンプルなPHPの図書管理システム
家にある本を写真に撮ってサイトにアップし、書名・作者・表紙・概要を保存して、自分や友達が見られるようにするのが目的の個人用図書管理プロジェクト。
今の主な機能はこんな感じ：
パスワードでサイトに入る：固定パスワードを入力しないと入れない
本のアップロード：表紙画像をアップして、書名・作者・概要を入力
本の一覧：すべての本の表紙、書名、作者を表示
本の詳細ページ：詳細情報や大きい画像を表示、編集と削除も可能
検索機能：書名や作者で本を検索できる
PHP+MySQL+HTML/CSSで作った小さな個人用図書整理サイト
時間があれば、今後はお気に入り（收藏）機能も追加する予定。例えば新しいテーブルを作って、お気に入りにした本を記録して、別のページで一覧表示できるようにするつもり。
